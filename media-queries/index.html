Notes

Common breakpoints for widths of devices:

  • 320px — 480px: Mobile devices (i.e. @media (max-width: 480px) {...} )
  • 481px — 768px: iPads, Tablets (i.e. @media (max-width: 768px) {...} )
  • 769px — 1024px: Small screens, laptops (i.e. @media (max-width: 1024px) {...} )
  • 1025px — 1200px: Desktops, large screens (i.e. @media (max-width: 1200px) {...} )
  • 1201px and more —  Extra large screens, TV (i.e. @media (min-width: 1201px) {...} )

Normally, in most of my projects I only use two media queries. One for phone widths and one for tablet width. The rest I just use the normal global styles.

But there is one that I'd like to use more such as the prefers-color-scheme. Which checks the prefered colour scheme on you device, such as 'light' or 'dark' mode on your device.
